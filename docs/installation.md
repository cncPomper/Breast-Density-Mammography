# Installation

## Stable release

To install Breast-Density-Mammography, run this command in your terminal:

```sh
uv add Breast-Density-Mammography
```

Or if you prefer to use `pip`:

```sh
pip install Breast-Density-Mammography
```

## From source

The source files for Breast-Density-Mammography can be downloaded from the [Github repo](https://github.com/cncPomper/Breast-Density-Mammography).

You can either clone the public repository:

```sh
git clone https://github.com/cncPomper/Breast-Density-Mammography
```

Or download the [tarball](https://github.com/cncPomper/Breast-Density-Mammography/tarball/main):

```sh
curl -OJL https://github.com/cncPomper/Breast-Density-Mammography/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd Breast-Density-Mammography
uv sync
```
